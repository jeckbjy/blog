# hugo使用测试

- 官网:https://gohugo.io/
- 中文:http://www.gohugo.org/

## 常用命令

- 生成站点:hugo new site /path/to/site
- 新建文档:hugo new about.md 
- 运行调试:hugo server --theme=hyde --buildDrafts
- 生成部署:hugo --theme=hyde --baseUrl="http://coderzh.github.io/"
